# Market Pulse Dashboard

Interactive financial dashboard built with Python and Streamlit.

## Features

- Market performance comparison
- Risk statistics
- Correlation analysis
- Rolling volatility
- Drawdown analysis
- Interactive asset selection
- Time range controls

## Technologies

- Python
- Streamlit
- Plotly
- pandas
- yfinance

## How to Run

pip install -r requirements.txt
streamlit run app.py
